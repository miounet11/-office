## V2 起点

V1.5 完成意味着**功能 + 中文 + 视觉低悬果三件套就位**。V2 应转向 **AI 一等公民**，按之前的"2026 AI 范式"规划：

| Wave | 内容 | 估算 |
|---|---|---|
| W1 | Provider runtime 壳 (本地 Ollama only) + AI eval 框架 | 2-4 周 |
| W2 | Cmd+K v0 (Writer 限定) — 自然语言 → SfxDispatcher slot fuzzy match (不接 LLM) | 2-3 周 |
| W3 | Writer 智能分析器 apply runtime (从 preview-only 升级) | 4-6 周 |
| W4 | Select-to-Act 浮窗 + Diff 审批视图 | 6-8 周 |
| W5 | 异步任务管理器 (Cowork 风格) — 周报/月报委托工作流 | 持续 |

**V2 完整规划已写入 `docs/product/v2-master-plan.md` 与 `docs/product/v2/` 子目录**：

- [`v2-master-plan.md`](v2-master-plan.md) — 总体架构、时间线、风险
- [`v2/w1-provider-runtime-spec.md`](v2/w1-provider-runtime-spec.md) — Provider runtime
- [`v2/w2-cmd-palette-spec.md`](v2/w2-cmd-palette-spec.md) — Cmd+K 命令调色板
- [`v2/w3-writer-apply-runtime-spec.md`](v2/w3-writer-apply-runtime-spec.md) — Writer Apply Runtime
- [`v2/w4-select-to-act-spec.md`](v2/w4-select-to-act-spec.md) — Select-to-Act 浮窗 + Diff 审批
- [`v2/w5-async-cowork-spec.md`](v2/w5-async-cowork-spec.md) — 异步任务管理器

V2 实施时各 wave 单独立项 (`.agent/goals/v2-w<N>-<topic>/`)。
